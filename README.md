# Project Overview
Educational institutions often require insights into student performance to provide timely support and improve outcomes. This project uses machine learning algorithms to predict student exam scores by analyzing factors such as gender, parental education, lunch type, test preparation, and previous scores.

The project covers the complete ML lifecycle:
Data analysis and visualization
Feature engineering
Model training and evaluation
Deployment using a Flask web application
